# Warm Up 2019 后端

## 技术栈
Node + Koa + MySQL

## 使用方式

- 安装依赖: `cnpm i`
- 启动调试服务: `npm run server`

## 目录结构

```
src
├── controller          // 用于解析用户的输入，处理后返回相应的结果
├── models              // 数据表模型
├── app.js              // 入口文件
├── config.js           // 配置文件
├── init.js             // 数据库初始化脚本
├── router.js           // 路由文件
package.json            // 项目配置文件
```
